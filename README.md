## Estrutura do Repositório

O repositório contém as implementações em `C++` desenvolvidas ao longo da disciplina, abrangendo diferentes desafios algorítmicos. Para facilitar a compreensão, **cada arquivo contém um cabeçalho (header) em formato de comentário no topo**, explicando brevemente o problema resolvido e a técnica ou algoritmo utilizado.

Os arquivos estão divididos em:
- `e*-assunto.cpp`: Exercícios práticos e problemas isolados para fixação de conceitos (o nome de cada arquivo carrega seu respectivo assunto após o prefixo).
- `t*-assunto.cpp`: Trabalhos mais complexos e pequenos projetos que exigem combinação de múltiplas técnicas (também nomeados de forma idiomática).

## Tecnologias

Todos os códigos foram desenvolvidos em **C++** (padrão C++11/C++14/C++17), tirando amplo proveito da **Standard Template Library (STL)**. Estruturas como `std::vector`, `std::map`, `std::set`, e `std::priority_queue` formam a base eficiente para muitas das soluções apresentadas, assim como os algoritmos embutidos na biblioteca `<algorithm>`.

## Técnicas e Algoritmos Abordados

Neste repositório, você encontrará soluções construídas com técnicas clássicas e avançadas de Ciência da Computação, tais como:

- **Estruturas de Dados:** Union-Find (Disjoint Sets), Trie (Árvore de Prefixos) e Fila de Prioridade (Min-Heaps e Max-Heaps).
- **Grafos:** Busca em Profundidade (DFS), Busca em Largura (BFS), Algoritmo de Kruskal (MST), Ordenação Topológica (Algoritmo de Kahn) e Fluxo Máximo (Ford-Fulkerson / Edmonds-Karp).
- **Programação Dinâmica:** DP 1D, DP 2D (Intervalos / Estilo Matrix Chain), Máquina de Estados e Memoization (DP Top-Down em grafos).
- **Divisão e Conquista:** Adaptações do Merge Sort (contagem de inversões), Cálculo de Convex Hull (Geometria Computacional) e Algoritmo para Par de Pontos Mais Próximos (Closest Pair).
- **Algoritmos Gulosos:** Activity Selection e ordenações especializadas (Radix Sort).
- **Matemática Computacional:** Exponenciação Modular Rápida, Testes de Primalidade e Inverso Multiplicativo Modular (Pequeno Teorema de Fermat).
