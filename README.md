## Estrutura do Repositório

O repositório contém as implementações em `C++` desenvolvidas ao longo de ambas as disciplinas, abrangendo diferentes desafios algorítmicos. Para facilitar a compreensão, **cada arquivo contém um cabeçalho (header) em formato de comentário no topo**, explicando brevemente o problema resolvido e a técnica ou algoritmo utilizado. Além disso, cada arquivo carrega o respectivo assunto como sufixo no próprio nome para facilitar a identificação.

Os arquivos estão divididos em:
- `e*-assunto.cpp`: Exercícios práticos e problemas isolados para fixação de conceitos (Teoria).
- `t*-assunto.cpp`: Trabalhos mais complexos e pequenos projetos que exigem combinação de múltiplas técnicas (Teoria).
- `Lab-Alg-Avancado/atividade*-assunto.cpp`: Implementações de desafios de programação competitiva e algoritmos avançados (Laboratório).

## Tecnologias e Técnicas Abordadas

Todos os códigos foram desenvolvidos em **C++** (padrão C++11/C++14/C++17), utilizando extensivamente os contêineres e algoritmos da **Standard Template Library (STL)**.

Neste repositório, você encontrará soluções construídas com técnicas clássicas e avançadas de Ciência da Computação, tais como:

- **Algoritmos Gulosos (Greedy):** Activity Selection, pareamento, problemas de agendamento (Job Scheduling) e ordenações personalizadas (Radix Sort).
- **Programação Dinâmica (DP):** 1D DP (Coin Change, Knapsack 0-1, Jump Game), 2D DP (Caminhos e somas máximas em grids, Intervalos / Estilo Matrix Chain), Máquina de Estados e Memoization (DP Top-Down em grafos), além de DP com Bitmask (TSP / Combinações).
- **Estruturas de Dados:** Union-Find (Disjoint Sets), Trie (Árvore de Prefixos), Fila de Prioridade (Min-Heaps e Max-Heaps) e Binary Indexed Trees (Fenwick Trees) para contagem de prefixos.
- **Grafos:** Busca em Profundidade (DFS), Busca em Largura (BFS), Componentes Fortemente Conexas (Kosaraju e Tarjan), Caminhos Mínimos (Dijkstra), Árvore Geradora Mínima (Kruskal), Ordenação Topológica (Algoritmo de Kahn) e Fluxo Máximo (Ford-Fulkerson / Edmonds-Karp).
- **Divisão e Conquista:** Adaptações do Merge Sort (contagem de inversões), Cálculo de Convex Hull (Geometria Computacional) e Algoritmo para Par de Pontos Mais Próximos (Closest Pair).
- **Backtracking:** Busca Exaustiva para resolução de restrições clássicas, como o Problema das N-Rainhas e o Solucionador de Sudoku.
- **Strings e Pattern Matching:** Algoritmo de Knuth-Morris-Pratt (KMP) para correspondência de substrings.
- **Matemática Computacional:** Exponenciação Modular Rápida, Testes de Primalidade, Princípio da Inclusão-Exclusão, Cálculo do MDC e MMC, e Algoritmo de Euclides Estendido para Inverso Multiplicativo Modular.
